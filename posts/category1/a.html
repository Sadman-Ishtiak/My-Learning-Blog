Getting Started with Vanilla JavaScript Modules

ES modules are built into modern browsers and Node.js. You can split your code into reusable pieces without any third-party tooling. Just use type="module" on your script tags and import/export syntax.

Example:

export function greet(name) {
  return `Hello, ${name}!`;
}

Then in another file:

import { greet } from './utils.js';
console.log(greet('World'));
